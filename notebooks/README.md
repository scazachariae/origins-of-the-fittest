# Notebooks

This directory contains example notebooks demonstrating the variant_dynamics package.

## Current Examples (Recommended)

- **`example_basic_simulation.ipynb`** – **START HERE**. Complete introduction covering:
  - Loading bundled networks
  - Running `simulation_phylogeny`
  - Visualizing results with Altair
  - Sampling approximations for inspection

- **`example_dmp_approximation_ba1024.ipynb`** – Dynamic message passing walkthrough using `si_dynamic_message_passing_unweighted` / weighted variants on a Barabási–Albert graph.

- **`example_shortest_path_approximation_ba1024.ipynb`** – Distance-based approximation and τ/H integral demo built from the same network, cross-checking deterministic spreading statistics.

## Getting Started

1. Install the package
2. Launch Jupyter/VS Code and open `example_basic_simulation.ipynb`
3. Use the main `README.md` for environment + dependency instructions
